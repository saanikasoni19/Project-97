The first thing you should know is what you do in HTML. Well, In HTMl, you add elements, like buttons, text, inputs, etc.

When you start coding in HTML, you should start with the HTML tag which looks like <html>. I call it opening html tag. At the end of all you code in the html file, you have to end with the ending html tag, which looks like </html>. I call it ending html tag. An OPTIONAL tag to put AFTER the opening html tag is the head tag with looks like this, <head>. Of course this also has and ending tag which looks like </head>. INSIDE the head tags, another OPTIONAL tag to put is the title tag which looks like <title>. You should ONLY put it inside the head tag. It also has a closing tag which looks like </title>. Not all of the tags in HTML have and ending tag. One thing to know isthat in between the opening and ending tag of html is where you put all of your code. So after the html beginning tag, or after the head closing tag, is where you will put body. The body is where you basically put the most important elements in like the text, and buttons, etc. It looks like this,<body></body>. Inside the body tags, you put in the text. There are seven different tags to put in text. The first six look like this. <h1></h1>, <h2></h2>, <h3></h3>, <h4></h4>, <h5></h5>, <h6></h6>. The h1 is the biggest, and the h6 is the smallest. The other text tag is the paragraph tag wihcih looks like this, <p></p>. In between the closing and opening of the text tag is where you'll put your text.

Here is an example of code.

<html>

<head>

<title>

Hello

</title>

</head>

<body>

<h1>

I am amazing

</h1>

</body>

</html>
